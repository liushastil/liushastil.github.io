|浏览量:

查看系统基本信息

  • 系统版本
    uname -a

  • 安装时间
    stat /root/anaconda-ks.cfg
    stat /lost+found/

  • 获取主机名
    cat /etc/hostname

系统目前安装的rpm

rpm -qa

rpm -V -a

  • 查看防火墙规则
    iptables-save > path

  • 查看当前的进程
    ps -aux

  • 查看加载的所有模块
    lsmod

网络信息

  • 网卡信息
    ip a

  • dns信息

cat /etc/resolv.conf

  • 路由表

route -n

  • arp表
    arp -a

  • 网络端口信息
    netstat -antpul

日志信息

/var/log/cron*
/var/log/mail*
/var/log/mysqld.log*
/var/log/yum.log*
等等
