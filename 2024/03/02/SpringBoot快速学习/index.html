参数传递

这个主要是使用@RequestParam注解

Get请求

@RequestMapping(value=”/test1”, method=RequestMethod.GET)

  • 接收传参的参数

POST请求

@RequestMapping(value=”/test2”, method=RequestMethod.POST)

  • 发送普通的参数和Get方法是一样的
  • 发送json体
    • 在方法中的括号里使用@RequestBody注解
    • public String test2(@RequestBody User user){…}

静态资源访问

在Idea中新建SPring项目后，默认会创建classpath:/static/目录，静态资源放在这个目录即可

  • 若是直接放到这个static目录下的话，直接在网站的根目录输入资源名称即可访问，一般是会新建一个虚拟路径来访问静态资源

例如：spring.mvc.static-path-pattern=/images/**

这个就表示在根目录下添加/images/就可以访问静态资源了

  • 新建静态资源文件目录，在resources目录下新建css文件夹

    spring.web.resources.static-locations=classpath:/css
