ffuf的使用

发表于2024-03-23|更新于2024-10-31|工具
|浏览量:

✍内容

下载地址

https://github.com/ffuf/ffuf/

直接去release中下载编译好的amd64的

使用教程

使用关键字来确定fuzz的位置，默认关键字是FUZZ，可以自己定义，只需要在字典的后面加上冒号+名称即可

帮助

ffuf -h

代理

-x socks5://127.0.0.1:10810

指定文件头

  • 指定User-Agent
    -H “User-Agent: Mozilla/5.0 Windows NT 10.0 Win64 AppleWebKit/537.36 Chrome/69.0.3497.100”

递归fuzz

  • 指定递归2层
    -recursion 2

匹配正则

-mr “sucess”

匹配状态码

  • 匹配需要的
    -mc 200，403
  • 匹配所有
    -mc all

匹配响应大小

-ms 1291

过滤响应文件大小

  • 指定大小为42的过滤掉
    -fs 42

正则过滤

-fr “error”

指定后缀

-e jsp,zip

输出

  • 若是没有输出就不创造文件
    -or
  • 输出指定格式
    -of md
  • 输出到文件
    -o result.md

  • 批量网站扫描敏感目录

  • 爆破findsomething插件找出的单个网站的路径是否存活
